# 📋 待办事项应用 (Todo App)

一个简洁现代的待办事项网页应用，使用纯 HTML + CSS + JavaScript 构建。

## 项目结构

```
test01/
├── index.html      # 主页面（HTML 结构）
├── css/
│   └── style.css   # 样式表（所有样式）
├── js/
│   └── app.js      # 主逻辑（所有交互）
└── README.md       # 项目说明
```

## 功能特性

- ✏️ **添加任务** — 输入内容后点击按钮或按 Enter 键
- ☑️ **标记完成** — 点击复选框切换完成状态
- ~~已完成任务显示删除线~~
- 🗑️ **删除任务** — 每个任务右侧有删除按钮
- 📊 **任务统计** — 实时显示全部 / 未完成 / 已完成数量
- 🧹 **清除已完成** — 底部按钮一键清除
- 💾 **本地存储** — 数据保存在 localStorage，刷新不丢失

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 语义化页面结构 |
| CSS3  | Flexbox 布局 + 渐变 + 动画 + 响应式 |
| JavaScript (ES5) | 原生 DOM 操作，无框架依赖 |
| localStorage | 浏览器本地数据持久化 |

## 设计特点

- 清新蓝色渐变主题
- 圆角卡片式布局
- 柔和阴影效果
- 自定义复选框动画
- 新任务滑入动画
- 移动端响应式适配

## 使用方法

1. 双击打开 `index.html` 文件
2. 或用浏览器直接打开该文件
3. 开始添加你的待办事项！
